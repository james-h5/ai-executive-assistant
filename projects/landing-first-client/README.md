# Landing First Client

Everything needed to go from zero to a signed first AI consulting client.

**Status:** Planning

## Description
The goal is to sign the first paying AI consulting client by end of 2026. This project tracks the strategy, outreach, and pipeline for making that happen.

## Key Dates
- Target: Signed by end of 2026

## Next Steps
- Define ideal client profile (industry, size, pain points)
- Decide on outreach channel (Instagram, cold outreach, referrals, Skool network)
- Build a simple offer (what's the first engagement look like?)
- Track pipeline here once outreach starts

## Channels
- **Direct outreach** — warm network + cold Instagram DMs to Brisbane trade businesses (primary)
- **Fiverr / Upwork** — secondary channel, running in parallel with outreach. Full strategy in [freelance-platforms-strategy.md](freelance-platforms-strategy.md), pipeline tracked in [freelance-pipeline.md](freelance-pipeline.md). Five sub-approaches, fastest to slowest:
  1. Freelance prompt-engineering / AI workflow setup gigs — $50-150/hr, fastest to first dollar
  2. Lightweight AI audits ($200-400, 2-3hrs) — doubles as a case study for the Brisbane-trades consultancy
  3. AI content creation for creators (scripts, SEO, email sequences) — lowest technical lift
  4. Productized templates ($97-297 each) — passive once built, needs marketing
  5. Agency white-labeling — partner with agencies needing AI implementation help
