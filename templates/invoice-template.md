# Invoice Template

Simple, plain-text invoice — no need for invoicing software until there are enough clients to
justify it. A clean PDF/doc from this template is fine for the first several clients.

---

## Invoice Structure

```
INVOICE

From: James Heathcote — [ABN if applicable]
To: [Business Name] — [Owner Name]
Invoice #: [YYYY-MM-NNN, e.g. 2026-07-001]
Date: [date]
Due: [date, per payment terms in the services agreement]

| Description | Amount |
|---|---|
| [System Name] — Setup (one-time) | $[amount] |
| [System Name] — Monthly fee ([month]) | $[amount] |
| **Total** | **$[amount]** |

Payment via: [bank transfer details / payment link]
Questions: [email/phone]
```

## Notes
- Setup fee invoice goes out on signing (or per the payment split agreed in the services agreement).
- Monthly invoices go out on a fixed date each month (e.g. the 1st) so it becomes a predictable
  rhythm for both sides.
- Keep a running log of invoices sent/paid somewhere (a simple table is enough at this scale) —
  don't rely on memory once there's more than one client.
