# Outreach Message Templates

Style guide for the `outreach` skill when drafting personalized cold outreach to Brisbane trade businesses. Casual Australian-tradie tone — these are tradies, not corporate executives. Every message follows the same shape: hook → pain question → 60-second auto-reply pitch → demo link → soft CTA.

Pick the template that matches the lead's structural pain signal (from the ClickUp task description), then personalize the `[hook]` line using the specific detail already captured for that lead (hours, contact method, review quote, etc.). Don't reuse a hook verbatim across leads — it should read like James actually looked at their site.

---

## Template A — On-the-tools / mobile only

*Use when: the business runs on a mobile number with no contact form, i.e. the owner is the point of contact and is usually on a job.*

> Hey [Name] — [hook, e.g. "saw you're based out in Redlands"].
>
> When you're mid-job and a quote request comes through — what happens to it?
>
> Built something that auto-replies within 60 seconds, asks the basic questions, and sends you a summary. Existing customers don't get the bot treatment — it recognises them and just lets you know they called.
>

> Worth a chat?

## Follow-up templates

Shorter and lighter than the initial message — don't re-pitch, just nudge. Used when a lead moves into `Follow-up 1` or `Follow-up 2`. Keep both to 3 short lines: greeting + reason for the nudge, pain-point restatement + ask, soft close. Use the owner's first name if the business name gives one (e.g. "Jason's Plumbing" -> "Jason"), otherwise "[Business] team".

**Follow-up 1** (a few days after the initial message, no reply yet):

> Hey [Name/team], just floating this back up in case it got buried.
> If [one-line restatement of the pain point, e.g. "the auto-reply idea is still relevant for those after-hours calls"], keen for a quick chat.
> No stress if it's not a priority right now — happy to send the demo through whenever suits.

**Follow-up 2** (final nudge, a bit more direct):

> Hey [Name/team], don't want to keep bugging you, so last one from me.
> If [one-line restatement of the pain point] is still relevant, keen for a quick chat.
> If not, all good — appreciate you having a look either way.

## Channel selection

- Instagram handle present → DM using Templates A/B/C (drop the "Subject:" framing, keep it short — IG DMs read worse when long)
- No Instagram handle → Template D (email)
- Pick A vs B vs C based on which structural pain signal is strongest in the lead's stored data (mobile-only takes priority over generic "manual quote process" if both apply)
